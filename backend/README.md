# Mapgenius Solutions – Backend API

This repository implements a **SaaS‑ready electronic invoicing platform** compatible with the Colombian DIAN (and extensible to other tax authorities).

## Core Features
- **Multitenant FastAPI** with JWT authentication.
- **CRUD** for Clients, Products/Services, and Invoices.
- **DIAN integration (simulated)** – UBL 2.1 XML generation, CUFE calculation, RSA signature, QR placeholder.
- **AI‑assisted validation** of invoice payloads using Pydantic.
- **Email delivery** with XML & PDF attachments.
- **Reporting** endpoints for dashboard visualizations.
- **PDF generation** via Jinja2 + WeasyPrint (fallback to HTML).

## Development Setup
1. **Clone the repo** and `cd` into `backend`.
2. **Python 3.11+** and `pip` installed.
3. Install dependencies:
   ```bash
   python -m pip install -r requirements.txt
   ```
4. Create a `.env` (copy from `.env.example`) and set `DATABASE_URL=sqlite:///mapgenius.db` for local dev.
5. Run Alembic migrations:
   ```bash
   alembic upgrade head
   ```
6. Start the API:
   ```bash
   uvicorn app.main:app --reload
   ```
   The docs are available at `http://localhost:8000/docs`.

- Unit & integration tests.
- Build a Docker image (optional).

## Extending to Real DIAN / SAT Integration
- Replace `app/services/dian_client.py` with the official SOAP/REST client.
- Store real certificates in a secure vault and adjust `utils/signature.py` to use PKCS#7 signatures.
- Add country‑specific tax codes in the `Product` model.
