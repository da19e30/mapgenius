# Mapgenius Solutions – Frontend UI

This repository contains the **React + TypeScript** frontend for the Mapgenius SaaS invoicing platform.

## Features
- **Modern SaaS look** – Tailwind CSS + Vite, responsive layout.
- **Authentication** with JWT (AuthContext). 
- **Clients & Products** management pages (CRUD via FastAPI).
- **Invoice Wizard** – multi‑step form to create DIAN‑compliant electronic invoices.
- **Dashboard** – Recharts visualizations for income, IVA, and invoice status.
- **Dark/Light theme toggle** (ThemeContext) and toast notifications.

## Development Setup
1. **Prerequisites** – Node.js ≥ 18, npm ≥ 9.
2. **Install dependencies**
   ```bash
   cd frontend
   npm install
   ```
3. **Environment variables**
   Create a `.env` (or `.env.local`) file at the project root with:
   ```env
   VITE_API_URL=http://localhost:8000/api/v1   # FastAPI base URL
   ```
4. **Run the dev server**
   ```bash
   npm run dev
   ```
   The app will be available at `http://localhost:5173`.

## Building for Production
```bash
npm run build
```
The optimized static assets are emitted to `dist/` and can be served by any static web server.

## Testing
- **Unit / component tests** – use Vitest (already configured). Run `npm test`.
- **End‑to‑end tests** – Cypress located in `cypress/`. Run `npm run cy:open`.

## Linting & Formatting
```bash
npm run lint   # ESLint
npm run fmt    # Prettier
```
